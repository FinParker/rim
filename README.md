# Rim

A small text-editor written by Rust.

一个使用Rust编写的小型文本编辑器.

- Build follow [build-your-own-x](https://github.com/codecrafters-io/build-your-own-x) and [hecto](https://philippflenker.com/hecto/)

- 我在学习Rust的过程中, 根据hecto的指引建立了这个文本编辑器.

ci: CI/CD配置
